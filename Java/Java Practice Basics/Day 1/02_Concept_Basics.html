1. Java Ecosystem & Setup

Welcome to Java!

The Big Three

  • JDK (Java Development Kit): The toolbox for developers. Includes 'javac' (compiler) and 'java' (runner).
  • JRE (Java Runtime Environment): The library needed to run Java programs.
  • JVM (Java Virtual Machine): The engine that executes bytecode. "Write Once, Run Anywhere!"

IntelliJ IDEA Setup

  1. Install JDK 17.
  2. Download IntelliJ IDEA Community.
  3. Create a New Project and write your first Main class.

Compile vs Run

javac Hello.java -> Generates Hello.class (Bytecode)
java Hello -> JVM runs the bytecode!

🏅 The Leap Year Rule

  • 1️⃣ Divisible by 4 → Normally a leap year.
  • 2️⃣ But NOT by 100 → Then it is NOT a leap year.
  • 3️⃣ Exception: Divisible by 400 → It IS a leap year again!
Logical Formula: (y%4==0 && y%100!=0) || (y%400==0)

🔹 08: Max of Three (Ternary)

public class MainClass {
  public static void main(String[] args) {
    int a=10, b=20, c=15;
    int max = (a > b) ? (a > c ? a : c) : (b > c ? b : c);
    System.out.println("Max is: " + max);
  }
}
