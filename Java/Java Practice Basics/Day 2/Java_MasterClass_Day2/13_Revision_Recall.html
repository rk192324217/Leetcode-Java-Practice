3. Mental Model Flashcards

Click a card to reveal the Core Concept.

Trial Division vs Sieve?
Trial Division ($O(N\sqrt{N})$) is for single numbers. Sieve ($O(N\log\log N)$) is for ranges.
Why check only up to $\sqrt{N}$?
Because factors appear in pairs. If no factor is found up to $\sqrt{N}$, none exist beyond it.
Pre-increment vs Post-increment?
Pre (++a): Change THEN Use.
Post (a++): Use THEN Change.
What is a Strong Number?
Sum of factorials of digits equals the number (e.g., 145 = 1! + 4! + 5!).
