6. Sieve of Eratosthenes

The most efficient way to find primes up to N. "Sieve" out multiples of each prime.

Click Start to Sieve Primes up to 50

⚡ Performance: Sieve vs. Trial Division

When finding all primes up to N = 1,000,000:

Metric Trial Division (Normal) Sieve of Eratosthenes
Strategy Check divisors for every number "Cross out" multiples of primes
Time Complexity O(N √N) O(N log log N)
Efficiency Slow for large N Blazing fast (Industry Standard)

🚀 For N=1M, Trial Division takes ~2-3 seconds, while Sieve takes < 0.1 seconds!

🔹 Digit Sum

public class MainClass {
  public static void main(String[] args) {
    int n=1234, s=0;
    while(n>0) {
        s += n%10;
        n /= 10;
    }
    System.out.println("Sum: " + s);
  }
}
