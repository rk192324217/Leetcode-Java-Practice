1. 🔢 Armstrong Numbers (1-999)
Find all numbers where sum of digits raised to power of digit count equals the number itself

💡 How it Works

  • Count the digits in the number (e.g., 153 has 3 digits)
  • Raise each digit to the power of digit count: 1³ + 5³ + 3³
  • If sum equals original number, it's Armstrong!

2. 🔄 Palindrome Number Check
Check if a number reads the same forwards and backwards (121 → true)

💡 How it Works

  • Reverse the number digit by digit
  • Compare reversed number with original
  • If they match, it's a palindrome!

4. ↔️ Reverse Array (Two Pointers)
Reverse array in-place using two pointers: [1,2,3,4,5] → [5,4,3,2,1]

💡 Two Pointer Technique

  • Place LEFT pointer at start (index 0)
  • Place RIGHT pointer at end (index length-1)
  • Swap elements at LEFT and RIGHT
  • Move LEFT forward, RIGHT backward
  • Continue until LEFT >= RIGHT

💻 Code Walkthrough
int[] arr = {1,2,3,4,5}; int left = 0; int right = arr.length - 1; while (left < right) { // Swap elements int temp = arr[left]; arr[left] = arr[right]; arr[right] = temp; left++; right--; }
Result:
Original: [1,2,3,4,5]
Reversed: [5,4,3,2,1]
⏱ Time: O(n)
💾 Space: O(1) - In-place reversal!
5. 🎯 Move All Zeros to End
Move zeros to end while maintaining order: [1,0,3,0,5] → [1,3,5,0,0]

💡 How it Works

  • Use pointer POS to track position for next non-zero
  • If element is non-zero, place it at POS and increment
  • After loop, fill remaining positions with zeros

6. 🌊 Kadane's Algorithm - Maximum Subarray Sum ⭐
Find contiguous subarray with largest sum: [-2,1,-3,4,-1,2,1] → max = 6

💡 The Stream Decision Concept

At each element, you face a choice:

  • Extend: Add this number to current sum (keep going)
  • Start Fresh: Begin a new subarray from here

current = max(element, current + element)

Rule: If current sum drags you down (negative), restart!

⏱ Time: O(n) - Single pass!
💾 Space: O(1) - Only two variables
7. 🎯 Two Sum (HashMap Approach)
Find two numbers that add to target: [2,7,11,15], target=9 → [0,1]

💡 HashMap Strategy

  • Store each number and its index in a HashMap
  • For each number, calculate complement = target - number
  • Check if complement exists in map
  • If found, return both indices!

⏱ Time: O(n) - Single pass
💾 Space: O(n) - HashMap storage
