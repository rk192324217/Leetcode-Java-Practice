🌊 Container With Most Water

LeetCode #11 - Interactive Visual Guide

📖 Understanding the Problem

The Water Container Analogy

Imagine: You have vertical walls at different positions with different heights. Find two walls that form a container holding the maximum water.

Key Points:

  • 🏗️ Water height = height of the shorter wall (water spills over!)
  • 📏 Width = distance between the two walls
  • 💧 Water Area = width × min(height[left], height[right])
Area = (j - i) × min(height[i], height[j])

where i = left pointer, j = right pointer

🤔 Why Move the Shorter Wall?

If you move the taller wall:

  • ❌ Width decreases
  • ❌ Height still limited by shorter wall
  • ❌ Area can only decrease!

If you move the shorter wall:

  • ✅ Width decreases BUT might find taller wall
  • ✅ Gives a chance for larger area!

💻 Java Code
public int maxArea(int[] height) { int i = 0, j = height.length - 1; int maxArea = 0; while (i < j) { int width = j - i; int minHeight = Math.min(height[i], height[j]); int area = width * minHeight; maxArea = Math.max(maxArea, area); if (height[i] < height[j]) { i++; } else { j--; } } return maxArea; }
